මෙම වැඩසටහන නිවැරදිව ක්‍රියාත්මක වීමට නම් ඔබ අනිවාර්යයෙන් මෘදුකංග ත්‍රිත්වයක් පරිගණකයට බාගත කර තිබිය යුතුය. ඒවා නම් වෙබ් බ්‍රව්සරයක්, සිංහල යුනිකේත මෘදුකාංගයක් හා ෆ්ලෑෂ් ධාවකයකි. මේ සඳහා දැනට හොඳ භාවිතයක් පවතින මෘදුකාංග ත්‍රිත්වයක් පහතින් දැක්වේ. එම මෘදුකාංග අන්තර්ජාල සම්බන්ධතාවයකින් තොරව ලබා ගත හැක.


මෘදුකාංග ලබා ගන්නා ආකාරය :

  • ඉහතින් ඇති අයිතම ත්‍රිත්වයෙන් අවශ්‍ය අයිතමය මත ක්ලික් කරන්න.
  • එවිට පැමිණෙන වින්ඩෝවේ දී Open File විධානය ලබා දෙන්න.
  • ඉන් පසු එම මෘදුකාංගය WinRAR හරහා විවෘත වේ. (WinRAR අවශ්‍ය වේ)
  • ඒ තුළ ඇති මෘදුකාංගයේ නම සහිත අයිතමය විවෘත කරගන්න.
  • සාමාන්‍ය ආකාරයට මෘදුකාංගයක් පරිගණක ගත කරන අයුරින් එය Install කර ගන්න.
